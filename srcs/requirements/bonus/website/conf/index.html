Démarrage sur le serveur
En tant que serveur, nginx est utilisé avec des paramètres minimaux pour les pages web statiques. Très net.
Télécharger sur Docker

Le projet est conteneurisé et déployé sur Docker utilisant Alpine 3.16. Et c'est une bonne chose !
